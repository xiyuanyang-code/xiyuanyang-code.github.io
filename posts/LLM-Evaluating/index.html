Preparing API keys

对于闭源大模型和缺乏硬件支持的玩家来说，使用API密钥进行LLM调用是非常方便且高效的选择。

对于国内玩家，非常推荐使用 智增增 作为API接口的第三方平台。在这个网站上申请得到API Key之后，我们就可以准备激活自己的LLM了。

首先，我们需要在当前目录下创建一个.env文件，用于后续读取OPENAI_API_KEY和BASE_URL，当然，你也可以直接写在环境变量之中，一劳永逸。
