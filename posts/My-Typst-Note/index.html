Typst

Typst Official Docs

天下苦 LaTeX 久矣！

Typst 作为新型的一种文档编译语言，兼具编译能力和编译性能，于我而言，现在几乎全面转战 Typst 的原因在于：

  • 作为 Rust 项目简单的安装和命令行接口，免去了 TeX 复杂和庞大的命令行安装流程
  • 简单简介的语法和语法糖
    • 数学公式和 LaTeX 保持基本一致，可移植性高
    • 并且作为一种编程语言，可读性更高（相信我，写多了你会爱上他的！）
  • 闪电般的编译速度
  • Vscode 给力的插件支持和 inline suggestions
    • 支持实时查看和快速 PDF 编译
    • 支持语法高亮，就像一门编程语言！
  • 第三方社区支持：
    • （相对还不错的）社区支持
    • 这一点还不能和 LaTeX 比，但是仍然在快速增长，并且我个人的使用需求这一点并非痛点！

在 Overleaf 一再缩减编译时间的情况下，日常场景下简单的笔记文档编译再使用 LaTeX 已经显着效率低下了！
