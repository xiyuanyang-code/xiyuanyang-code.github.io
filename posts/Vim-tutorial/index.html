Abstract

This document provides a comprehensive guide to the basic functionalities of Vim, a powerful text editor. It covers essential commands and modes, including Command Line, Normal Mode, Visual Mode, and Insert Mode. The guide also includes common shortcuts and tips for efficient text editing, such as moving within documents, searching, copying, cutting, and pasting text. Whether you’re a beginner or looking to refine your Vim skills, this document serves as a valuable resource for mastering Vim’s core features.

Introduction

What is Vim?

Modes in Vim

Command Line

Command mode can be entered by typing : in Normal mode. Your cursor will jump to the command line at the bottom of the screen upon pressing :. This mode has many functionalities, including opening, saving, and closing files, and quitting Vim.

  • :q quit (close window)
  • :w save (“write”)
  • :wq save and quit
  • :e {name of file} open file for editing
  • :ls show open buffers
  • :help {topic} open help
    • :help :w opens help for the :w command
    • :help w opens help for the w movement

Normal Mode

  1. Document Movement Sequence

    • Vertical Movement (gg, G, ^u, ^d, HEL)
    • Horizontal Movement (0, ^, $, w, b, e)
    • High-Precision Movement (HJKL)

  2. Content Search/Jump (%, f, F, t, T, /)

  • Basic movement: hjkl (left, down, up, right)
  • Words: w (next word), b (beginning of word), e (end of word)
  • Lines: 0 (beginning of line), ^ (first non-blank character), $ (end of line)
  • Screen: H (top of screen), M (middle of screen), L (bottom of screen)
    • This command allows for quick document navigation.
  • Scroll: Ctrl-u (up), Ctrl-d (down)
  • File: gg (beginning of file), G (end of file)
    • You can also use ]] (top) and [[ (end) commands.
  • Line numbers: :{number}<CR> or {number}G (line {number})
  • Misc: % (corresponding item)
    • Automatically jumps to the first matching point in the current file (e.g., between parentheses () or brackets []).
  • Find:
    • Find/to forward/backward {character} on the current line.
    • , / ; for navigating matches.
  • Search: /{regex}, n / N for navigating matches.

Visual Mode

  • Visual: v
  • Visual Line: V
  • Visual Block: Ctrl-v
  • d, c, y for delete, change, and yank (copy) operations.

In Vim, Visual mode provides an intuitive way to select text. Here are the three main Visual modes and their uses:

1. Visual Mode (v)

  • Enter: Press v to enter Visual mode.
  • Use: Selects text at the character level.
  • Example: Move the cursor to select text, then use operators like d (delete), c (change), or y (yank) on the selected text.

2. Visual Line Mode (V)

  • Enter: Press V (Shift + v) to enter Visual Line mode.
  • Use: Selects entire lines of text.
  • Example: Move the cursor up or down to select multiple lines, then perform line-level operations like deleting (d) or copying (y).

3. Visual Block Mode (Ctrl-v)

  • Enter: Press Ctrl + v to enter Visual Block mode.
  • Use: Selects a rectangular block of text, useful for editing tables or performing the same operation on multiple lines.
  • Example: Select a rectangular area, then insert (I or A), change (c), or delete (d) text within the block.

General Operations:

  • Exit Visual Mode: Press Esc or Ctrl + [ to exit any Visual mode.
  • Extend Selection: Continue moving the cursor to expand the selection.
  • Shrink Selection: Press o to switch the cursor to the other end of the selection, then move the cursor to shrink the selection.
  • Operate on Selected Text: Once text is selected, use standard Vim operators like d (delete), y (yank), or c (change) on the selected text.

Pasting in Visual Mode:

  1. Select Text: Enter Visual mode (v, V, or Ctrl + v) and select the target area.
  2. Copy Text: In Normal mode, use y to yank (copy) the selected text.
  3. Paste Text: In Visual mode, press p to paste after the selection or P to paste before the selection.

4. Insert Mode

Everything that you used to do with the mouse, you now do with the keyboard using editing commands that compose with movement commands. Here’s where Vim’s interface starts to look like a programming language. Vim’s editing commands are also called “verbs”, because verbs act on nouns.

  • i Enter Insert Mode
    • For manipulating/deleting text, use more than just backspace.
  • o / O insert line below / above (in Normal mode)
  • d{motion}
    • e.g., dw is delete word, d$ is delete to end of line, d0 is delete to beginning of line.
  • c{motion}
    • e.g., cw is change word.
    • Like d{motion} followed by i.
  • x delete character (equal to dl)
  • s substitute character (equal to cl)
  • Visual mode + manipulation
    • Select text, d to delete it or c to change it.
  • u to undo, <C-r> to redo
  • y to copy / “yank” (some other commands like d also copy)
  • p to paste
  • ~ flips the case of a character.

Counts

You can combine nouns and verbs with a count, which will perform a given action a number of times.

  • 3w move 3 words forward
  • 5j move 5 lines down
  • 7dw delete 7 words

Modifiers

You can use modifiers to change the meaning of a noun. Some modifiers are i, which means “inner” or “inside”, and a, which means “around”.

  • i - “inner” or “inside”, acts on the inside of brackets, quotes, etc.
    • ci( change inside parentheses.
    • ci[ change inside square brackets.
    • ci" change inside double quotes.
    • ci' change inside single quotes.
  • a - “around”, includes the brackets, quotes, etc.
    • ca( change around parentheses.
    • ca[ change around square brackets.
    • ca" change around double quotes.
    • ca' change around single quotes.

Common Shortcuts

Undo in Normal Mode

  • u to undo the last operation.
  • Ctrl + r to redo the last undo.

Select All Text / Copy / Cut in Visual Mode

  • Select All Text: In Normal mode, go to the start (gg) or end (G) of the document, then enter Visual mode and select the text.
  • Copy: Use y to yank (copy) the selected text.
  • Cut: Use d to delete (cut) the selected text.

Save / Exit in Command Line

  • :w to save.
  • :q to quit.
  • :wq to save and quit.

Cut in Normal Mode (x, dd, D, d)

  • Cut Single Character: x deletes and cuts the character under the cursor.
  • Cut Entire Line: dd deletes and cuts the entire line.
  • Cut to End of Line: D deletes and cuts from the cursor to the end of the line.
  • Cut in Visual Mode: Select text in Visual mode and press d to cut the selected text.

This document provides a foundational understanding of Vim’s core functionalities, enabling users to navigate, edit, and manipulate text efficiently. By mastering these commands and modes, you can significantly enhance your productivity when working with Vim.
