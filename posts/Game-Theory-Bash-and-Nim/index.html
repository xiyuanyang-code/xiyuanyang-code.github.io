Game Theory: Bash and Nim

Game Theory: Bash and Nim

Game Introduction

现在进行一个游戏，两位玩家 A 和 B 分别一次从一堆球堆中取球，获胜规则是取完所有球堆中所有球的玩家获胜，具体的规则如下：

  • 总球堆 S 被分为若干子球堆 (s1,s2,…,sn)(s_1, s_2, \dots, s_n)(s1​,s2​,…,sn​), n 为总球堆的总个数，而 sis_isi​ 为一个正整数。
  • 每次拿球的时候每个人允许拿的球数是 [1,k][1, k][1,k]，不可以多拿也不可以不拿。
  • 每次拿球只允许从一个子球堆中取球，不允许在一次取球中取多个子球堆中的球。

根据策梅洛定理，表示在二人的有限游戏中，如果双方皆拥有完全的信息，并且运气因素并不牵涉在游戏中，那先行或后行者当中必有一方有必胜/必不败的策略。

问题：假设初始球堆状态 SSS 给定，A 为先手，B 为后手，在初始球堆状态 SSS 为何种情况下，A 存在必胜策略？（反正 B 存在必胜策略，因为这个游戏不存在平局）。从整体来看，谁的赢面更大？

数学理论分析

显然，对于一种特殊情况，如果当前每一个球堆的球的个数都是 k+1k+1k+1 的倍数，那后手存在必胜策略。

更进一步的，推导致一般的情况，定义初始异或和：

G=(s1(modk+1))⊕(s2(modk+1))⊕⋯⊕(sn(modk+1))G = (s_1 \pmod{k+1}) \oplus (s_2 \pmod{k+1}) \oplus \dots \oplus (s_n \pmod{k+1}) G=(s1​(modk+1))⊕(s2​(modk+1))⊕⋯⊕(sn​(modk+1))

我们阐述定理：

  • 当且仅当初始异或和 G≠0G \neq 0G=0 时，先手必胜。
  • 当且仅当初始异或和 G=0G = 0G=0 的时候，后手必胜。

异或运算的性质

  • 归零性： A⊕A=0A \oplus A = 0A⊕A=0
  • 恒等性： A⊕0=AA \oplus 0 = AA⊕0=A
  • 交换律与结合律： A⊕B=B⊕AA \oplus B = B \oplus AA⊕B=B⊕A。
  • 自反性： 如果 A⊕B=CA \oplus B = CA⊕B=C，那么 C⊕B=AC \oplus B = AC⊕B=A。

引理证明

我们首先需要证明如下三条定理：

  • 终局状态： 当所有球被取光时，g1=g2=⋯=gn=0g_1 = g_2 = \dots = g_n = 0g1​=g2​=⋯=gn​=0，此时异或和 G=0G = 0G=0。根据规则，最后取球的人获胜。（易证）

  • 从 G≠0G \neq 0G=0 到 G=0G = 0G=0 的可达性： 只要 G≠0G \neq 0G=0，玩家总能通过某种合法动作修改球堆状态使得新的异或和变为 000。

  • 从 G=0G = 0G=0 到 G≠0G \neq 0G=0 的必然性： 如果当前 G=0G = 0G=0，任何合法的取球动作，都会导致新的异或和 G′G'G′ 不为 0。

引理 2 证明

我们已知初始状态的异或和为 GGG：

(g1⊕⋯⊕ 除去 gi⊕⋯⊕gn)=G⊕gi(g_1 \oplus \dots \oplus \text{ 除去 } g_i \oplus \dots \oplus g_n) = G \oplus g_i (g1​⊕⋯⊕ 除去 gi​⊕⋯⊕gn​)=G⊕gi​

(G⊕gi)⊕gi′=0(G \oplus g_i) \oplus g_i' = 0 (G⊕gi​)⊕gi′​=0

根据异或运算的最后一条性质，我们可以推导出：gi′=gi⊕Gg_i' = g_i \oplus Ggi′​=gi​⊕G

注意：上述式子中的 iii 是任取的，因此关键在于找到一个 iii，使得 gi→gi′g_i \to g_i'gi​→gi′​ 的操作是合法的！

证明目标： 找到一堆 gig_igi​，使得将其改变为 gi′=gi⊕Gg_i' = g_i \oplus Ggi′​=gi​⊕G 后，新的异或和为 000，且该操作合法（即 gi′<gig_i' < g_igi′​<gi​）。

新的异或和为 0 已经在上述式子中证明。

设 GGG 的二进制表示中，最高位的 111 在第 mmm 位。
根据异或定义，既然 GGG 的第 mmm 位是 111，那么在所有堆 {g1,g2,…,gn}\{g_1, g_2, \dots, g_n\}{g1​,g2​,…,gn​} 中，必然存在至少一堆 gig_igi​，其第 mmm 位也是 111（否则 111 的个数为偶数，异或后该位应为 000）。我们令 gi′=gi⊕Gg_i' = g_i \oplus Ggi′​=gi​⊕G。

比较 gig_igi​ 和 gi′=gi⊕Gg_i' = g_i \oplus Ggi′​=gi​⊕G 的二进制位：

  • 对于高于第 mmm 的位：因为 GGG 在这些位上全是 000，所以 gi′g_i'gi′​ 和 gig_igi​ 在这些位上完全相同。
  • 对于第 mmm 位：gig_igi​ 该位是 111，GGG 该位也是 111，异或结果 gi′g_i'gi′​ 在该位变为 000。
  • 对于低于第 mmm 的位：无需比较。

因此，gi′<gig_i' < g_igi′​<gi​ 必然成立。玩家只需从第 iii 堆中取走 (gi−gi′)(g_i - g_i')(gi​−gi′​) 个球即可实现。并且因为 gi=(si(modk+1))∈[0,k]g_i = (s_i \pmod{k+1}) \in [0, k]gi​=(si​(modk+1))∈[0,k]，因此此次取球符合条件！

从算法上来说，玩家只需要挑选一个球堆，其余数 gig_igi​ 的最高位的 1 和当前 GGG 的最高位的 1 所在的位数相同（这样的球堆保证存在），并且从该球堆中取出计算好数量的球，就可以保证调整后的 G′≠0G' \neq 0G′=0。

引理 3 证明

初始状态 g1⊕g2⊕⋯⊕gn=0g_1 \oplus g_2 \oplus \dots \oplus g_n = 0g1​⊕g2​⊕⋯⊕gn​=0。假设玩家操作了第 jjj 堆，将其数量从 gjg_jgj​ 变为 gj′g_j'gj′​。

先手必胜的局面

当且仅当初始异或和 G≠0G \neq 0G=0 时，先手必胜。

  • 假设初始 G≠0G \neq 0G=0：
  • 第一次博弈：
    • A 面对 G≠0G \neq 0G=0，他可以执行一个动作，将局面变成 G=0G = 0G=0 留给后手。
    • B 面对 G=0G = 0G=0，留给 A 的局面必然是 G≠0G \neq 0G=0。
  • 循环：只要游戏没结束，A 总是拿到 G≠0G \neq 0G=0 并把它变成 000；B 总是拿到 000 并把它变成非 000。
  • 收敛性：由于每次操作球的总数都在严格递减，而球数不能为负，游戏必须在有限步内结束。
  • 根据定理 1，最终的结束局面必定为 G≠0G \neq 0G=0 的情况，因此此时先手 A必胜。

后手必胜的局面

当且仅当初始异或和 G=0G = 0G=0 的时候，后手必胜。

证明过程完全类似，可以使用策略窃取的想法，先手 A 的第一步棋无论如何会导致G≠0G \neq 0G=0，此时先后手交替，B 获胜。

实验验证

Dynamic Programming

我们也可以使用动态规划进行高效的穷举计算：

  • 首先，我们对初始球堆进行取余操作（问题是等价的）
  • 接下来，对于表示的状态 S，我们定义 dp 状态：
    • dp[S][0] 代表当前状态下，为 A 的拿球轮次，是否存在 A 的必胜策略？1 for true, 0 for false.
    • dp[S][1] 代表当前状态下，为 B 的拿球轮次，是否存在 A 的必胜策略？1 for true, 0 for false.
  • 状态转移：
    • dp[S][0] 的计算: 如果存在一个可达的状态 S′S'S′，如果 dp[S'][1] 为 1，我们就认为 dp[S][0] = 1（注意，只需要找到可达状态就可以）
    • dp[S][1] 的计算：如果对于每一个可达状态 S′S'S′，dp[S'][0] 均为 1，则认为 dp[S][1] = 0（要穷尽全部的可达状态）
  • 最终判断初始状态下 dp[S_0][0] 是否为 1？
  • 简单情况：定义结束状态 S*（球已经被取完），dp[S*][0] = 0, dp[S*][1] = 1。

Results

胜负的关键在于随机生成的初始化球堆状态，对于一个完全随机生成的球堆，很显然 A 的赢面远大于 B，换句话说，在随机初始化的条件下，先手具有绝对的先发优势。

时间复杂度分析

  • DP: O(状态数×n×K)O(\text{状态数} \times n \times K)O(状态数×n×K)，哪怕在一开始已经做了取余的操作，随着每次可以拿球数量的快速上升，状态数成指数爆炸式增长，时间复杂度高。
  • 理论解法：只需要求解一个异或和即可，时间复杂度为 O(n)O(n)O(n)

About Game Theory

巴什博弈 (Bash Game)

规则：有一堆 nnn 个物品，两个人轮流取，每次取 [1,m][1, m][1,m] 个，取走最后一个的人获胜。

必胜态：如果 n(modm+1)≠0n \pmod{m+1} \neq 0n(modm+1)=0，先手必胜；否则后手必胜。

尼姆博弈 (Nim Game)

规则：有 nnn 堆物品，每堆数量任意。每次可以从某一堆中取走任意数量（最少 1 个，最多不限），取走最后一个的人获胜。

核心逻辑：通过异或运算计算所有堆的“尼姆和”（Nim-sum）。

G=s1⊕s2⊕⋯⊕snG = s_1 \oplus s_2 \oplus \dots \oplus s_n G=s1​⊕s2​⊕⋯⊕sn​

必胜态：如果 G≠0G \neq 0G=0，先手必胜。

本游戏本质上就是巴什博弈和尼姆博弈的混合体，计算方式也是天然的相似。
